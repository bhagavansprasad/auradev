- Login to pgsql shell 'with user postgress'
$ su - postgres

- Reset admin password
  # \password postgres

- Create new user 'gerrit'

- Set new password 'jnjnuh' to new user
  # ALTER USER gerrit with PASSWORD 'jnjnuh';
